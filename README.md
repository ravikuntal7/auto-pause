# Auto Pause Chrome Extension

A simple and efficient Chrome extension that automatically pauses any playing video when you switch tabs and resumes it when you switch back.

## Features

-   **Auto-Pause:** Instantly pauses videos (YouTube, Vimeo, etc.) when the tab becomes inactive.
-   **Auto-Resume:** Automatically resumes playback when you return to the tab.
-   **Lightweight:** Minimal impact on browser performance.

## Installation

Since this extension is not yet in the Chrome Web Store, you need to install it in **Developer Mode**:

1.  Clone or download this repository.
2.  Open Google Chrome and go to `chrome://extensions`.
3.  Enable **Developer mode** in the top right corner.
4.  Click on the **Load unpacked** button.
5.  Select the **`auto-pause-extension`** folder from this repository.

## Usage

1.  Open any website with a video (e.g., YouTube).
2.  Start playing the video.
3.  Switch to another tab -> The video will pause.
4.  Switch back to the video tab -> The video will resume.
